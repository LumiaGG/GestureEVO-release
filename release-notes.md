# GestureEVO 2.1-0410001

更新范围：自 `260324001`（`bb7be01e9d055e54a2389d9e169734439c40c88b`）以来

本次更新是一轮较大规模的功能整合，重点覆盖 Inspire 图片处理、外部调用链路、搜索体验、设置页可用性，以及整体性能与稳定性。相比 `260324001`，这次版本不仅加入了全新的 EVO 图片编辑器，也重新梳理了外部调用授权与链接打开逻辑，并补充了更多面向日常使用的细节配置。

## 新增

- 新增 EVO 图片编辑器，可在 Inspire 流程中直接进行裁剪、涂鸦、形状标注、撤销/重做等操作，并整合保存与分享流程。
- 新增外部调用授权体系，支持授权弹窗、授权记录管理、免确认调用记录，以及更清晰的调用来源展示。
- 新增设置页搜索，可直接搜索并跳转到对应设置项，减少层层翻找。
- 新增应用别名功能，常用应用可自定义显示名称。
- 新增应用选择器拼音搜索，查找中文应用时更方便。
- 新增手势时间/阈值设置，可调节长按、双击、滑动停顿、折返距离等关键识别参数。
- 新增搜索历史记录持久化与相关管理能力。
- 新增浏览器 Custom Tabs 支持，部分网页类搜索可使用更轻量的打开方式。

## 优化

- 重做 Inspire 图片处理链路，补齐编辑器启动动画、控制点触摸区域、笔刷大小显示，以及分享按钮行为一致性。
- 优化图片搜索、分词页面、搜索页面与设置页面的整体交互体验。
- 优化搜索结果排序逻辑，应用结果会更优先展示匹配度更高、使用更频繁及最近使用的项目。
- 优化“添加/编辑搜索引擎”页面布局与配置方式，直接链接、外部跳转、分享等模式区分更清晰。
- 优化默认链接打开方式与搜索引擎链接配置流程，减少配置成本。
- 优化外部调用链接结构，重构为更清晰的内部授权模式，并补充轮盘跳转按钮等辅助入口。
- 优化轮盘与触发器配置，补充更多触发参数与形状相关设置。
- 优化关于页、Inspire 设置页、QuickTools 与多处文案细节。

## 修复

- 修复 OCR 与无障碍文本获取切换时的异常问题。
- 修复 OCR 排序错误。
- 修复异步赋值后模式按钮状态未刷新的问题。
- 修复 PictureFloating 创建过程中的潜在问题。
- 修复数据备份错误。
- 修复部分搜索输入与底部输入模式相关的交互问题。

## 性能与稳定性

- 优化 OCR 模型加载流程，减少识别前等待时间，提升响应速度。
- 优化悬浮窗内存占用，降低长时间使用下的资源压力。
- 优化通知型轮盘图标缓存机制，改善图标加载与切换流畅度。
- 引入 `ManagedBitmap` 管理位图所有权，降低内存泄漏风险。
- 优化 Inspire 与相关悬浮组件性能，提升连续使用时的稳定性。
