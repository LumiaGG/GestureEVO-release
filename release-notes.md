# GestureEVO 2.1-0410001

本次更新主要围绕搜索体验、搜索引擎配置流程、OCR 加载效率与悬浮窗性能进行了优化，整体使用体验更加流畅稳定。

## 更新内容

- 优化搜索结果排序逻辑，应用搜索结果会更优先展示匹配度更高、使用更频繁及最近使用的项目。
- 优化“添加/编辑搜索引擎”页面布局与交互，配置结构更清晰，操作更直观。
- 新增浏览器 Custom Tabs 支持，部分网页类搜索可使用更轻量的打开方式。
- 优化 OCR 模型加载流程，减少识别前的额外等待，提升响应速度。
- 优化悬浮窗内存占用，降低资源消耗，提升长时间使用下的稳定性。
- 优化通知型轮盘图标缓存机制，改善图标加载与切换流畅度。
